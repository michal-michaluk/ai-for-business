# Example: Propozycja wzbogacenia Kosztorys.xlsx (brief)

This is the *output of step 02* from a real enrichment run for an AI SDLC cost estimate.

## Proposed changes

### 1. New sheet: "Offer Overview"
Cover page with sections: Exec Summary, Approach & Phases, Scope per domain, Business Outcomes, Delivery Model, Risks, Why Bottega.

### 2. New column in "Podsumowanie": Opis oferty (G)
Extended narrative description for each TOTAL.x row.

### 3. New column in detail sheets: Value proposition (I)
Per-item description of what changes for each role (Analyst, Developer, Reviewer, QA).
