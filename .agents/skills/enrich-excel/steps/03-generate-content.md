# 03 — Generate narrative content

Write all narrative texts based on the approved brief.

## Input

- `brief.md` from step 02
- Source materials (PDF, website)

## Process

1. Generate content for each section defined in the brief:
   - **Executive Summary** — 3-5 sentences: what, why, value, total investment
   - **Approach & Phases** — methodology description (Audit → PoV → Scale)
   - **Scope descriptions** — per TOTAL.x area, narrative explanation
   - **Business outcomes** — time-to-market, tech debt, knowledge retention, etc.
   - **Delivery model** — cooperation forms, environments
   - **Risk mitigations** — hallucination, quality, security, vendor lock-in
   - **Why company** — credentials, clients, team
   - **Value proposition per item** — what each line item means for each role (Analyst, Developer, Reviewer, QA)

2. Write in the client's language (Polish or English)

## Output

`content.md` — all texts ready to be inserted into Excel.

## User approval required

- Review and approve all content before applying to Excel
- User may request tone/emphasis adjustments
