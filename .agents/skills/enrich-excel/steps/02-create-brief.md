# 02 — Create brief

Define what narrative enrichment is needed and where to apply it.

## Input

- `structure.md` from step 01
- Source material (PDF, website URL, client presentation, email)

## Process

1. Extract key messages from source material:
   - Executive summary points
   - Approach / methodology description
   - Business outcomes
   - Risk mitigations
   - Company credentials
2. Map content to Excel structure:
   - Which sheets get narrative columns
   - Where the Offer Overview sheet should go
   - What the target audience needs (Board vs Middle Management)
3. Propose specific changes

## Output

`brief.md` — a proposal with:
- New sheets / columns to add
- Content structure per section
- Target audience per section

## Approvals built in

- User confirms the scope before any content is written
- Avoid wasted work on wrong direction
